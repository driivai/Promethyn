# Specification Changelog

This file tracks changes to the protocol specification (`protocol.md` and
`invariants.md`). It is versioned independently from the implementation.
Changing an invariant requires the process described in `GOVERNANCE.md`.

## [0.1] — unreleased

### Added
- Initial protocol definition: entities, roles, the baseline run, and one
  learning cycle.
- Invariants I1–I5: held-out firewall, reversibility, auditability, verifier
  authority, and determinism of the reference loop.
- Conformance criteria tied to the bundled benchmark.
